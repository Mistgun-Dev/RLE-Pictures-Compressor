Shannon-Fano coding archiver

To compile the project:

$ make

To start the program:

$ ./src/shannon-fano

Shannon-Fano coding archiver by Minh Ngo (nlmin.nosp@m.htl@.nosp@m.gmail.nosp@m..com)

Ussage: ./shannon-fano OPTION input output -p compressing -u uncompressing

Example

    $ cat makefile

    main:
            gcc main.c bintree.c bitop.c pack.c sort.c unpack.c -o shannon-fano
